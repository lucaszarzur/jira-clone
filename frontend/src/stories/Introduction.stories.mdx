# Welcome to Angular Jira Clone Storybook

## What is Storybook

Storybook helps you build UI components in isolation from your app's business logic, data, and context.
That makes it easy to develop hard-to-reach states. Save these UI states as **stories** to revisit during development, testing, or QA.

## What is Angular Jira Clone Storybook

This is the collection of components for the Jira Clone project, includes:

- Avatar
- Breadcrumbs
- Button
- Input
- More to come...

## What is Angular Jira Clone

There have been a handful of cool Jira-cloned apps written in `React`/`VueJS`, which makes me wonder **Why not Angular**? And here you go.
This is not only a simplified Jira clone built with Angular, Akita and ng-zorro in the frontend and Java (Spring Boot), MySQL and Hibernate in the backend, but also an example of a **modern**, **real-world** full-stack application.

## Tech stack

![Tech Stack][stack]

### Frontend
- [Angular CLI][cli]
- [Akita][akita] state management
- UI modules:
  - [TailwindCSS][tailwind]
  - Angular CDK [drag and drop][cdkdrag]
  - [ng-zorro][ng-zorro] UI component: `tooltip`, `modal`, `select`, `icon` and more.
  - [ngx-quill][quill]
- [Netlify][netlify] for frontend deployment

### Backend
- [Java][java]
- [Spring Boot][spring-boot]
- [MySQL][mysql]
- [Hibernate/JPA][hibernate]
- [Docker][docker]
- [Heroku][heroku] for backend deployment
